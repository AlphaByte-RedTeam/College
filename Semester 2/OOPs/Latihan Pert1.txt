3. Class adalah blue print daripada sebuah object/suatu kumpulan dari beberap
   object/container yang menyimpan objek yang berhubungan dengan classnya

4. Object adalah code yang melakukan fungsi/tugas tertentu.

5. Abstraction, yaitu informasi relevan yang dibutuhkan untuk tugas tertentu
   Encapsulation, pengelompokkan suatu methods/behaviout tertentu, contohnya
		  kalkulator, pengelompokkan fungsi kalkulasi terbagi 4, 
		  yaitu +, -, *, /. Fungsi kalkulasi itu disimpan di dalam class
		  Calculate.
   Inheritance, yaitu sifat pewarisan Super Class yang bisa diwarisi ke Sub Classnya.
		Contoh, Class hewan, memiliki methods eat() dan sleep(), sifat ini
		bisa diwarisi ke Sub Class nya seperti Sub class Reptile, ada sifat
		eat(), sleep(), dan crawl(). Sub Class Mammal juga eat(), sleep(),
		breastfeed(), kemudian Human, eat(), sleep(), breasfeed(), speak(),
		walk(). 
		Jadi dari Super Class diwarisi ke Sub Classnya, dan dari Sub Class
		nya bisa mewarisi lagi sifat Sub Class sebelumnya
   Polymorphism, yaitu menggunakan method yang sama untuk melakukan berbagai task.
		 Contoh, Class Shape, memiliki method draw(), method ini bisa dipakai
		 untuk menggambar bentuk segitiga, kotak, atau lingkaran
